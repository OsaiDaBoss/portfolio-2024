Stack implemented on FPGA Board

This project implements a stack using an FPGA Board. A stack is a data structure that supports two main operations: Push, which adds an element to the stack, and Pop, which removes the most recently added element. The implementation involves the use of Verilog, registers, and seven-segment displays. The circuit consists of various components, including inputs for push and pop operations, as well as switches to determine the input values. Pushing a value onto the stack is done by enabling the push switch and pressing a control button. The stack operates through different states, allowing for pushing and popping elements, while LED lights indicate overflow or underflow conditions. The stack's logic is defined in Verilog code, which includes different states and adjusts the writing address based on the operation being performed. The values pushed onto the stack are stored in a register file, and the output from the register file is displayed on the seven-segment display through decoders. The project includes a finite state design that governs the different states of the program. The FPGA Board is used to run the project, with switches and buttons assigned specific functionalities, such as pushing, popping, resetting, and enabling or disabling the display.
